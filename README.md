# Backtracking and Constraint Propagation
A simple backtracking and constraint propagation implementation in Python for AI course.

The program was built to resolve the games of N-Queens and all types of Sudoku.

## Run
To run the program, start the ```Main.py``` file and follows the instructions on screen.

In the file ```sudoku.txt``` there are some sudoku example of different levels that the program can solve. 
To solve other types of sudoku, follow the format of those in the file.
